# 🎬 BiliSummary - AstrBot B站视频解析插件

一款为 [AstrBot](https://github.com/Soulter/AstrBot) 打造的 B 站视频深度解析插件。支持自动识别视频链接并生成精美的信息卡片。

![logo](logo.png)

## ✨ 功能特性
* 🚀 **多格式支持**：支持识别标准 `BV` 号及 `b23.tv` 短链接。
* 📺 **画质显示**：自动识别视频最高分辨率（1080/1440/2160）。
* 📝 **智能简介**：自动截断长简介并添加省略号，保持界面整洁。
* 📊 **数据统计**：实时显示播放、弹幕、投币及分享数。

## 🛠️ 安装方法
在 AstrBot 中发送以下指令：
`/plugin install https://github.com/meimingziyongle/astrbot_plugin_bili_summary`
